# Use Help
## 1. Basic operation(use Prefix expression)
```
eg.
    + 1 2 
    * 1 2 3 4 5
    / (* 10 10)(+ 1 1 1)
```
## 2. S-expression
```
eg.
    (+ 1 2)
```
## 3. Q-expression(it won't calculate)
```
eg.
    {+ 1 2}
    {1 2 (+ 5 6) 4}
```
